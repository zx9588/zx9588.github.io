    1. 安装node.js

    点击左边的下载后安装，一路默认。

    打开Mac终端，输入 sudo su 和密码

    2. 安装hexo

    为了提升安装速度，我们可以先安装cnmp

    输入以下指令：

    npm install -g cnpm --registry=https://registry.npm.taobao.org

    显示这样就代表下载完成了，那个warn不用管的

    然后输入以下指令，安装hexo

    cnpm install -g hexo-cli

    这样几秒钟就下完了

    3. 建立文件夹

    终端输入 mkdir blog

    就是建立了一个名叫‘blog’的文件夹，这个名字可以随便改

    然后输入cd blog/，进入这个文件夹

    输入pwd，就可以查看blog的路径

    sh-3.2# mkdir blog
sh-3.2# cd blog/
sh-3.2# pwd
/Users/zhangxin/blog

    4. 初始化生成博客

    输入sudo hexo init，当出现Start blogging with Hexo！就代表下载完成

    5. 启动博客

    输入 hexo s

    然后访问：http://localhost:4000/ 完成

    关闭的时候按 control + c 即可

    6. 博客操作相关

    新建博客 hexo n "我的第一篇博客"

    得到 INFO Created: /Users/zhangxin/blog/source/_posts/我的第一篇博客.md

    前往该路径 /Users/zhangxin/blog/source/_posts

    7. 将博客部署到github

    进入到自己的github，然后点右上角的加号，第一个new repository，

    名字必选是你的名字+’.github.io’，如我的就是：Chelseatr.github.io

    这是用户部署个人博客的GitHub仓库的命名要求

    Mac终端输入 cnpm install --save hexo-deployer-git下载git的插件

    再设置：vim _config.yml中的最底部，设置Deployment

    deploy:
  type: git
  repo: https://github.com/zx20219898/zx20219898.github.io.git
  branch: main

    再hexo d就可以了

    现在你就可以通过你gtihub的名字 + .github.io来访问
    比如我的博客: https://zx20219898.github.io/

    输入i 开始编辑

    编辑好esc按键 然后输入：wq 退出

    仓库地址: https://github.com/zx20219898/zx20219898.github.io

    8. 3-hexo 主题配置

    参考
    https://yelog.org
    https://github.com/yelog/hexo-theme-3-hexo

    10. 备份技巧

    把blog目录进行github备份本地项目上传GitHub

    为常用指令配置别名

    alias hs='hexo clean && hexo g && hexo s'
alias hd='hexo clean && hexo g && hexo d'
alias hds='hexo clean && hexo g && hexo d && hexo s'
alias hdd='hexo clean && hexo g && hexo d && git add . && git commit -m "update" && git push -f'

    11. Hexo常用命令

    hexo new postName       #新建文章 在source/_posts 下生成
hexo new page pageName  #新建页面 在source/pageName 下生成
hexo deploy             #将.deploy目录部署到GitHub 简写指令：hexo d 
hexo generate           #生成静态页面至public目录 简写指令：hexo g 
hexo server             #开启预览访问端口（默认端口4000，'ctrl + c'关闭server）简写指令hexo s 
hexo clean              #清除所有静态页面和缓存数据

    hexo clean && hexo g && hexo d 
hexo clean && hexo g && hexo s

hexo clean && hexo g && hexo d && git add . && git commit -m "update" && git push -f
