    Hexo安装

    • 参考1
    • 使用hexo+github搭建免费个人博客详细教程-好记的博客
    • 个人github仓库地址

    1. 安装node.js

    点击左边的下载后安装，一路默认。

    打开Mac终端，输入 sudo su 和密码

    2. 安装hexo

    为了提升安装速度，我们可以先安装cnmp

    输入以下指令：

    npm install -g cnpm --registry=https://registry.npm.taobao.org

    显示这样就代表下载完成了，那个warn不用管的

    然后输入以下指令，安装hexo

    cnpm install -g hexo-cli

    这样几秒钟就下完了

    3. 建立文件夹

    终端输入 mkdir blog

    就是建立了一个名叫‘blog’的文件夹，这个名字可以随便改

    然后输入cd blog/，进入这个文件夹

    输入pwd，就可以查看blog的路径

    sh-3.2# mkdir blog
sh-3.2# cd blog/
sh-3.2# pwd
/Users/zhangxin/blog

    4. 初始化生成博客

    输入sudo hexo init，当出现Start blogging with Hexo！就代表下载完成

    5. 启动博客

    输入 hexo s

    然后访问：http://localhost:4000/ 完成

    关闭的时候按 control + c 即可

    6. 博客操作相关

    新建博客 hexo n "我的第一篇博客"

    得到 INFO Created: /Users/zhangxin/blog/source/_posts/我的第一篇博客.md

    前往该路径 /Users/zhangxin/blog/source/_posts

    7. 将博客部署到github

    进入到自己的github，然后点右上角的加号，第一个new repository，

    名字必选是你的名字+’.github.io’，如我的就是：Chelseatr.github.io

    这是用户部署个人博客的GitHub仓库的命名要求

    Mac终端输入 cnpm install --save hexo-deployer-git下载git的插件

    再设置：vim _config.yml中的最底部，设置Deployment

    deploy:
  type: git
  repo: https://github.com/zx20219898/zx20219898.github.io.git
  branch: main

    再hexo d就可以了

    现在你就可以通过你gtihub的名字 + .github.io来访问
    比如我的博客: https://zx20219898.github.io/

    输入i 开始编辑

    编辑好esc按键 然后输入：wq 退出

    仓库地址: https://github.com/zx20219898/zx20219898.github.io

    Hexo常用命令

    hexo new postName       #新建文章 在source/_posts 下生成
hexo new page pageName  #新建页面 在source/pageName 下生成
hexo deploy             #将.deploy目录部署到GitHub 简写指令：hexo d

    第二步那个页脚 LOGO，如果你是 Butterfly 主题的话，可以直接在主题的配置文件找到 custom_text 并加入以下代码

    Vercel托管

    Vercel官网 使用 GitHub 登录。（比较难进，被墙）

    如果你想绑定自己的域名，就在输入框中输入你想要绑定的网址，点击右侧 Add 即可。下方会提示 Invalid Config，然后在域名购买方控制台进行域名解析。解析完成后即可通过自己的域名访问自己的博客。

    • 如果是根域名，请在你的 DNS 解析处添加 A 记录，指向 76.76.21.21
    • 如果是子域名，请在你的 DNS 解析处添加 CNAME 记录，指向 cname.vercel-dns.com

    docker运行hexo镜像

    • docker环境使用篇
    • https://hub.docker.com/r/taskbjorn/hexo
    • Docker部署hexo
    docker pull taskbjorn/hexo

docker volume create my_hexo_data
docker run -it -f-name my_hexo_container -p 4000:4000 -v my_hexo_data:/home/hexo/.hexo taskbjorn/hexo

    数据卷位置：/var/lib/docker/volumes

    换服务器时搬运

    进入容器内部:

    docker exec -it my_hexo_container sh

    exit：退出容器
