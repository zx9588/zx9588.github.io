为常用指令配置别名

Linux
发布时间 : 2022-07-21 18:47
阅读 :

  1. 查看shell echo $0 如果你的 shell 是zsh，则配置文件为.zshrc，如果是-bash 则为.bashrc
  2. cd ~/.zshrc 如果没有此文件，直接vi新增 vi .zshrc i 粘贴 ：wq
alias hs='hexo clean && hexo g && hexo s'
alias hd='hexo clean && hexo g && hexo d'
alias hds='hexo clean && hexo g && hexo d && hexo s'
alias hdd='hexo clean && hexo g && hexo d && git add . && git commit -m "update" && git push -f'
alias git-log='git log --pretty=oneline --all --graph --abbrev-commit'
  1. 执行source ~/.zshrc生效
